<h2 id="english">🇬🇧 English</h2>

A modern, multi-language web application built to help users seamlessly manage their income, expenses, and custom wallets. The system features a responsive Glassmorphism UI, real-time currency conversions via API, and interactive data visualization.

### 🚀 Core Features

- **Custom Wallets:** Manage specific accounts separately (Credit Card, Cash, Business).
- **Live Exchange Rates:** Automatically fetch live USD/EUR/TRY rates to calculate your net balance dynamically in different currencies.
- **Multilingual UI:** Complete dynamic support for English, Turkish, and Arabic (RTL architecture).
- **Interactive Charts:** Beautiful data visualization using Chart.js to track categorical income vs. expenses.
- **Saving Goals:** Set financial deadlines and track your progress percentages dynamically with a visual progress bar.
- **Export Data:** Print or export full transaction histories and filtered statements to PDF.
- **Admin Dashboard:** Total system overview, user control management, and site-wide statistics.
- **Secure Authentication:** Complete login/register system with secure password hashing and session timeout security.

### 🛠️ Tech Stack

- **Backend:** PHP (PDO, OOP Principles, Secure Session Management)
- **Frontend:** Vanilla CSS3 (Glassmorphism, Responsive Design), HTML5, JavaScript
- **Database:** MySQL
- **APIs & Libraries:** ExchangeRate API, Chart.js

### 🧠 Developer's Note & The Role of AI

My main goal in building this project was to put my skills into practice and see firsthand how database architecture, security fundamentals, and server management work together in the background.

Additionally, I integrated **Artificial Intelligence (AI)** during the development phase specifically to test its limits, pros, and cons in real-world software engineering. 

**My conclusion:** AI is an incredibly powerful tool that can dramatically speed up development and produce professional results. However, developers who use it blindly without grasping the underlying logic will ultimately lose out in the future. As a true software engineer, it is absolutely essential to understand exactly where every line of code comes from and what it does.

<h2 id="türkçe">🇹🇷 Türkçe</h2>

Kullanıcıların gelirlerini, giderlerini ve özel cüzdanlarını sorunsuz bir şekilde yönetmelerine yardımcı olmak için geliştirilmiş modern, çok dilli bir web uygulamasıdır. Sistem; duyarlı bir Glassmorphism (Cam tasarımı) arayüzüne, API üzerinden gerçek zamanlı döviz kuru dönüştürmeye ve etkileşimli veri görselleştirmelerine sahiptir.

### 🚀 Temel Özellikler

- **Özel Cüzdanlar:** Belirli hesapları ayrı ayrı yönetin (Kredi Kartı, Nakit, İş).
- **Canlı Döviz Kurları:** Net bakiyenizi farklı para birimlerinde dinamik olarak hesaplamak için canlı USD/EUR/TRY kurlarını otomatik olarak çeker.
- **Çok Dilli Arayüz:** İngilizce, Türkçe ve Arapça (RTL mimarisi) dilleri için tam destek.
- **Etkileşimli Grafikler:** Kategorik gelir ve giderleri takip etmek için Chart.js kullanan veri görselleştirmesi.
- **Birikim Hedefleri:** Finansal son tarihler belirleyerek ilerleme çubuğu ile başarı yüzdenizi anlık takip edin.
- **Veri Dışa Aktarımı:** Tüm işlem geçmişini ve filtrelenmiş hesap dökümlerini yazdırın veya PDF olarak dışa aktarın.
- **Yönetici Paneli (Admin):** Tüm sisteme genel bakış, tam kullanıcı yönetimi ve site geneli istatistikler.
- **Güvenli Kimlik Doğrulama:** Güvenli şifreleme (Password Hash) ve oturum zaman aşımı güvenliğine sahip tam kapsamlı giriş/kayıt sistemi.

### 🛠️ Kullanılan Teknolojiler (Tech Stack)

- **Backend:** PHP (PDO, OOP Prensipleri, Güvenli Oturum Yönetimi)
- **Frontend:** Vanilla CSS3 (Glassmorphism, Responsive Tasarım), HTML5, JavaScript
- **Veritabanı (Database):** MySQL
- **API ve Kütüphaneler:** ExchangeRate API, Chart.js

### 🧠 Geliştiricinin Notu ve Yapay Zekanın (AI) Rolü

Bu projeyi geliştirirken temel amacım, kişisel becerilerimi pratiğe dökmek ve veritabanı mimarisi, güvenlik temelleri ve sunucu (hosting) yönetiminin arka planda nasıl birlikte çalıştığını bizzat kendi ellerimle görmekti.

Ayrıca, geliştirme aşamasında yapay zekayı (AI), gerçek dünya yazılım mühendisliğindeki sınırlarını, artılarını ve eksilerini test etmek amacıyla projeme entegre ettim.

**Çıkarımım:** Yapay zeka, geliştirme sürecini inanılmaz derecede hızlandırabilen ve profesyonel sonuçlar üretebilen çok güçlü bir araçtır. Ancak, arka plandaki mantığı kavramadan onu körü körüne kullanan yazılımcılar gelecekte kesinlikle kaybedeceklerdir. Gerçek bir yazılım mühendisi olarak, her satır kodun tam olarak nereden geldiğini ve ne işe yaradığını anlamak kesinlikle zorunludur.
